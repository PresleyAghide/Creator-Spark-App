# Creator Spark

Creator Spark is a Progressive Web App built using Ionic and Angular.

## Features
- View content ideas from an external API
- Save and delete favorites using Ionic Storage
- View idea details on a separate page
- Settings page with two-way data binding
- Personalized greeting using saved user data
- Haptic feedback using Capacitor plugin

## Technologies
- Angular (Standalone Components)
- Ionic Framework
- HttpClient & Observables
- Ionic Storage
- Capacitor Haptics

## How to Run
npm install
ionic serve
